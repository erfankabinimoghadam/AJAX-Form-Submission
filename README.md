# AJAX-Form-Submission
Submit forms without page reload using AJAX. Supports dynamic recipient emails and provides real-time success or error feedback to users. Ideal for interactive contact or inquiry forms.

## Usage
1. Add `class="js-ajax-form"` to your form.
2. Optionally set `data-recipient="email@example.com"` for form-specific recipients.
3. Add an element with class `js-response-message` to display the feedback.
